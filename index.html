Facial Gestures Based Control System for Differently-abled Individuals

An advanced approach that encompasses an ensemble of regression tree models that allows differently abled individuals to access computers using dlib face detector to detect facial landmarks and thus predict facial gestures to control computers.

GitHub

Energy Auditing in households using Machine Learning and IoT

Created an comprehensive IoT and ML based system which measures the usage of power in households in real-time and the collected data is fed into an ML regression pipeline which is used to predict power consumption of upcoming months and recommendations are made to the users to reduce usage.

GitHub

Criminal face recognition and logging details using Deep Learning

Developed an Hybrid System which includes CAFFENet and 128-D(dimension) Facial landmark package in order to identify and recognize the criminal’s identity and log the live location and the time-stamp of the recognition activity from distributed CCTV systems.
